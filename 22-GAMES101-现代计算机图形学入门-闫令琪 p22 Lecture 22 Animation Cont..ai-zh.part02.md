### 一、 模拟方法：欧拉视角与拉格朗日视角

在数值模拟和计算机图形学中，观察流体或物体运动有两种核心视角，且常被结合使用。

1.  **拉格朗日视角/ 质点法**
    *   **核心逻辑**：盯着“物体”看。跟踪每一个质点或粒子随时间的运动轨迹。
    *   **关注点**：物体本身的位置移动。

2.  **欧拉视角/ 网格法**
    *   **核心逻辑**：盯着“空间”看。将整个空间划分为固定的网格，观察网格内属性随时间的变化，而不关心特定物体进出。
    *   **类比**：想象空间被切成横条格子，你盯着固定的格子，看不同时刻鸟（粒子）飞入或飞出该格子。
    *   **关注点**：固定空间位置上物理量（如密度）的变化。

3.  **混合方法**
    *   **Material Point Method (MPM) / 物质点方法**：
        *   **结合方式**：同时利用欧拉和拉格朗日视角。
        *   **具体流程**：
            1.  每个粒子携带材质属性（如粘性、密度、质量）。
            2.  在网格上进行模拟计算（如融化过程）。
            3.  计算完成后，将网格上的信息“写回”到对应的粒子上。
        *   **应用**：这是物理仿真领域非常火热的话题。

2.  **关于 OpenGL 等_API_的教学理念**
    *   课程中未直接教授 OpenGL 等具体 API，原因在于：
        *   学习底层原理（光栅化、变换、着色器等）后，自学 API 非常容易。
        *   掌握原理后，甚至可以手写软件渲染器。反之，若只会 API 而不懂原理，难以深入理解。

### 六、 讲师个人研究介绍与未来课程预告

讲师对自己在渲染领域的研究进行了详细梳理，并预告了可能的未来课程。

1.  **渲染的终极目标**
    *   ** Ultimate Realism（无限真实）**：让虚拟生成的画面达到人眼无法分辨真假的境界。

2.  **研究路线图**
    *   **基础**：实时与离线渲染技术。
    *   **已知问题**：如全局光照的加速。
    *   **未知问题**：外观建模。
    *   **新型显示挑战**：VR/AR 等头戴式设备带来的高帧率、双目渲染及注视点渲染等挑战。
    *   **前沿结合**：将当下热门技术应用于渲染。

3.  **预告课程**
    *   **《实时高质量渲染》**：
        *   内容：预计算、基于图像的渲染 (IBR)、非真实感渲染 (NPR，如《塞尔达传说》风格)、实时全局光照、光线追踪、DLSS 超采样等。
