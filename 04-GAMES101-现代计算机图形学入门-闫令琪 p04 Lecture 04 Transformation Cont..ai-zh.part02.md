### 1. 投影变换的前置知识与其他变换

*   **立方体标准化与拉伸问题**：
    *   在图形学流水线中，原本的长方体视景体最终会被缩放成一个标准立方体（通常范围为 -1 到 1）。
    *   这个过程会导致物体被拉伸。
    *   **视口变换**：在做完投影变换、物体进入标准立方体之后，还会执行一次视口变换。视口变换会再次对物体进行拉伸，从而修正之前的长宽比问题，确保最终显示的物体比例正确。因此，中间的拉伸不是问题。
*   **物体变换 vs 坐标系变换**：
    *   讲师个人倾向于使用“变换物体”来解释，而不是“变换坐标系”。
    *   变换坐标系（从一个坐标系变到另一个，导致物体坐标变化）本质上与变换物体一样，但前者不直观、难以理解。为了讲解方便，课程中主要以变换物体为主。
*   **输入与精度**：
    *   投影的输入（即视景体范围）是人为规定好的，决定了空间中哪一块区域需要显示。
    *   关于精度问题（Z-fighting等），讲师表示如果时间允许会后续提及，但本节课暂不展开。

### 2. 正交投影简介

*   **定义**：正交投影是相对简单的一种投影方式。
*   **形状**：在空间中定义一个长方体，包含近（Near）平面和远平面，物体直接沿平行线投影到平面上，没有“近大远小”的效果。

### 3. 透视投影

*   **特性与应用**：
    *   **广泛应用**：这是图形学应用中最常用的投影方式。
    *   **近大远小**：满足近处的物体大、远处的物体小的性质。
    *   **灭点现象**：原本平行的线在透视投影下不再平行，而是会汇聚于一点（灭点）。
        *   *例子*：照片中的铁轨。在欧几里得几何中，平行线永不相交；但在透视投影的照片中，两条平行的铁轨明显在远处相交于一点。这是因为三维空间被投影到了二维平面上，导致了视觉上的变化。
        *   如果用正交投影生成铁轨图片，平行线将永不相交，反而会让人觉得不自然。
*   **数学基础回顾：齐次坐标**：
    *   齐次坐标的定义：点 $(x, y, z)$ 可以表示为 $(x, y, z, 1)$。
    *   **缩放性质**：齐次坐标乘以任意非零常数 $k$，即 $(kx, ky, kz, k)$，表示的仍然是三维空间中的同一个点 $(x, y, z)$。
    *   *例子*：$(1, 0, 0, 1)$ 和 $(2, 0, 0, 2)$ 表示同一个点。这一性质是后续推导透视投影矩阵的关键。

### 4. 透视投影的核心策略：挤压变换

*   **化繁为简的思路**：
    *   直接写出透视投影的矩阵比较困难且不易理解。
    *   **解决方法**：将透视投影过程拆解为两步：
        1.  **挤压**：将原本的透视视景体（一个棱锥台/截头锥体，视频中称为 **Frost**，即 Frustum）压扁成一个长方体（即正交投影的视景体）。
        2.  **正交投影**：对这个压扁后的长方体进行标准的正交投影。
*   **“挤压”操作的定义与约束**：
    *   为了将截头锥体变为长方体，需要满足以下规定：
        1.  **近平面不变**：近平面上的所有点，在挤压过程中坐标完全不发生变化（即 $z$ 值不变，且 $x, y$ 也不变）。
        2.  **远平面 $z$ 不变**：远平面上的点，其 $z$ 值保持为 $f$ 不变，但其 $x, y$ 坐标会向中心收缩。
        3.  **中心点不变**：整个视景体的中心轴上的点（如 $x=0, y=0$ 的点）在挤压后位置不变。

### 5. 推导“挤压”矩阵

*   **利用相似三角形**：
    *   从侧面看视景体，任意一点 $(x, y, z)$ 经过挤压后，其 $y$ 坐标会变为 $y'$。
    *   根据相似三角形原理：$y' / y = n / z$ （其中 $n$ 是近平面的距离）。
    *   推导公式：$y' = (n / z) * y$。
    *   同理，对于 $x$ 坐标：$x' = (n / z) * x$。
    *   由此可知，挤压后的新坐标主要依赖于 $z$ 值。
*   **利用齐次坐标构建矩阵**：
    *   目标是找到一个 $4 \times 4$ 矩阵 $M$，使得 $M \cdot [x, y, z, 1]^T$ 的新结果能对应到上面推导的缩放关系。
    *   观察目标变换形式：
        *   我们希望得到的结果等效于 $[n \cdot x/z, \; n \cdot y/z, \; ?, \; z]$。
        *   为了消除分母 $z$，利用齐次坐标的性质，若整体乘以 $z$，则变为 $[n \cdot x, \; n \cdot y, \; ?, \; z^2]$ 的形式。
    *   由于这一步推导针对的是任意的 $x, y, z$，我们可以利用矩阵乘法逆推矩阵元素：
        *   **第一行**：要得到 $n \cdot x$，该行必须是 $[n, 0, 0, 0]$。
        *   **第二行**：要得到 $n \cdot y$，该行必须是 $[0, n, 0, 0]$。
        *   **第四行**：要得到 $z$，该行必须是 $[0, 0, 1, 0]$。
        *   **第三行**：目前未知，记为 $[0, 0, A, B]$ （因为前两元素如果非零会引入 $x$ 或 $y$ 的依赖，但 $z$ 的变换应只与 $z$ 有关）。
*   **求解第三行的未知数 $A$ 和 $B$**：
    *   需要利用前面提到的“挤压”约束条件。
    *   **条件一：近平面 ($z=n$) 不变**。
        *   输入点 $[x, y, n, 1]$ 经过矩阵变换后，结果必须表示同一个点。
        *   代入已知行和未知行，变换结果为：$[nx, ny, (An+B), n]$。
        *   为了表示同一个点，齐次坐标 $[nx, ny, (An+B), n]$ 必须与 $[x, y, n, 1]$ 成比例关系。为了消除 $x, y$ 系数 $n$ 的影响，整体除以 $n$ 归一化。
        *   归一化后第三项变为 $An/n + B/n = A + B/n$。
        *   要求结果 $z$ 坐标仍为 $n$，即 $A + B/n = n$。
        *   由此得到方程：$An + B = n^2$ ①。
    *   **条件二：远平面中心点 ($x=0, y=0, z=f$) 不变**。
        *   输入点 $[0, 0, f, 1]$ 变换后应表示点 $[0, 0, f, 1]$。
        *   变换结果为：$[0, 0, (Af+B), f]$。
        *   同样进行归一化，除以 $f$。
        *   归一化后第三项变为 $A + B/f$。
        *   要求结果 $z$ 坐标仍为 $f$，即 $A + B/f = f$。
        *   由此得到方程：$Af + B = f^2$ ②。
    *   **解方程组**：
        *   联立 ① 和 ②：
            1. $An + B = n^2$
            2. $Af + B = f^2$
        *   两式相减消去 $B$，得 $A(n-f) = n^2 - f^2$, $A(n-f) = (n-f)(n+f)$。
        *   解得：$A = n + f$。
        *   将 $A$ 代回方程 ①：$(n+f)n + B = n^2 \rightarrow n^2 + nf + B = n^2$。
        *   解得：$B = -nf$。
    *   **结论**：矩阵的第三行为 $[0, 0, n+f, -nf]$。
*   **最终矩阵**：
    *   这个将透视体挤压为正交体的矩阵称为 **Perspective to Orthographic** 矩阵。
    *   完整的透视投影矩阵 = 挤压矩阵 $\times$ 正交投影矩阵。

### 6. 课后思考题

*   **问题描述**：
    *   我们之前规定：对于近平面 ($z=n$) 和远平面 ($z=f$) 上的点，经过挤压变换后其 $z$ 坐标是不变的。
    *   **问题**：对于处于中间深度的点，例如 $z = (n+f) / 2$，经过挤压变换后，它的 $z$ 值会变小、变大还是不变？
*   **提示**：
    *   该点的 $z$ 值**不会**保持不变。
    *   考虑到 $n$ 和 $f$ 的方向（假设看向 z 轴负方向，或者 $n < f$ 等具体坐标系设定），思考这个点是会被“推”向近平面还是“吸”向远平面？
    *   建议课后将 $z$ 的显式变换公式写出来（通过推导出来的矩阵除以 $w$ 分量），观察其变化规律。
